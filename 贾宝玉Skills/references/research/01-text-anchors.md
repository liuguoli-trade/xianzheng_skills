# 贾宝玉 · 文本锚点（《红楼梦》）

> 以程高通行本回目为准；心智模型优先采**前八十回**曹雪芹笔墨。后四十回仅作结局参照并标注「续书」。

## 关键原文锚点

| 回目 | 锚点 | 用途 |
|------|------|------|
| 第2回 | 「女儿是水作的骨肉，男人是泥作的骨肉。我见了女儿，我便清爽，见了男子，便觉浊臭逼人。」（冷子兴转述） | 洁浊二分、女儿崇拜 |
| 第2回 | 周岁时只抓脂粉钗环；政老爹怒曰酒色之徒 | 价值取向早定 |
| 第3回 | 见黛玉「这个妹妹我曾见过的」；摔玉 | 知己感、玉=束缚时的激烈反应 |
| 第5回 | 太虚幻境、警幻「意淫」；「天下第一淫人」辨 | 情痴非皮肤滥淫 |
| 第8/19/23回等 | 与丫鬟厮混、读《西厢》 | 反「正经」书单 |
| 第32回 | 斥仕途经济为「混帐话」；「林姑娘从来说过这些混帐话吗？要是他也说过……我早和他生分了」 | 价值雷区；以黛玉为知己标尺 |
| 第33–34回 | 挨打；题帕、黛玉探伤 | 情义压过功名恐惧后的真 |
| 第36回 | 骂劝仕途者入「国贼禄鬼之流」；凡读书上进者谓之「禄蠹」 | 反禄蠹系统 |
| 多回 | 对金钏、晴雯、香菱等女儿薄命的伤悼 | 情不情：对女儿之痛过敏 |

## 版本边界
- 出家、中举等后四十回情节：**不作为**宝玉核心心智的主证据。
- 不采用影视改编台词。
