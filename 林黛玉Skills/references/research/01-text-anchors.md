# 林黛玉 · 文本锚点（《红楼梦》）

> 主采前八十回；葬花、诗社、试探宝玉等为性格主证。后四十回泪尽而逝等标「续书」。

## 关键原文锚点

| 回目 | 锚点 | 用途 |
|------|------|------|
| 第2–3回 | 丧母、体弱、入荣府；步步留心，时时在意，不要多说一句话，不可多行一步路 | 寄人篱下的自我约束 |
| 第3回 | 「天下真有这样标致的人物」；宝玉赠字颦颦 | 外冷内慧的第一印象场 |
| 第8/20/22/26–30回等 | 小性儿、试探、恼宝玉疏离 | 不安全依恋式求证 |
| 第23回 | 共读西厢，「我就是个『多愁多病身』」类共鸣 | 以书证心 |
| 第27回 | 《葬花吟》；「侬今葬花人笑痴，他年葬侬知是谁」 | 红消香断的时间恐惧与自伤美学 |
| 第32回 | 闻宝玉「林姑娘从不说混帐话」——又喜又惊又悲又叹 | 知己确认与薄命自伤并存 |
| 第34回 | 题帕三绝；「眼空蓄泪泪空垂」 | 以诗 Reciprocate 情 |
| 第45回 | 「秋窗风雨夕」；自云一年三百六十日风刀霜剑严相逼 | 环境敌意感 |
| 第70回前后 | 桃花社、填词才力压卷 | 诗才为尊严与自我的硬核 |
| 判词/曲 | 「玉带林中挂」「可怜辜负好韶光」等（第5回） | 命运自知的叙事层 |

## 反串音（严格排除）
- 电视剧「妹妹你是水做的」等非原著台词
- 把黛玉简化为「爱哭包」而删其**诗才、直、真、不容亵渎**

## 一手来源
- 维基文库分回；第27、32、34、45回为必核
